## 説明
最初、「悲愴三国志」として開発を始めましたが、大きい仕様で作っていたので、完成するか不透明さを感じ、  
コンパクト版として悲愴三国志Zeroをまず作って公開することとしました  
細かく区切って公開する方針とし、ソースコードもオープンとしました  
少しずつ拡張し、3作で1セットの構想にしています  
現在、悲愴三国志Zeroに3バージョン内包、悲愴三国志Zero2に1バージョン
